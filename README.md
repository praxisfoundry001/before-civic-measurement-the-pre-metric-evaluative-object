# Before Civic Measurement: The Pre-Metric Evaluative Object

## What is this?
A foundational civics bridge artifact in the Structural Flow library.

## Why should I care?
This artifact asks what must exist before civic measurement, preservation, comparison, or scoring can lawfully begin.

## What is the claim?
Before a civic-legal object can be preserved, compared, or scored, it must already have become one determinate evaluative object.

The pre-metric evaluative object is the already-constituted object that later preservation and later measurement presuppose rather than create.

## What is this not?
Not a full civic calculus.
Not a score system.
Not a worksheet.
Not a replacement for constitutional law, legal judgment, civic judgment, or domain expertise.

## How was this done?
The artifact builds on *Before Civic Evaluation* and asks the sharper next question: where does the required singleness of the evaluative object become real?

It distinguishes the pre-metric evaluative object from:
- binding content
- operative meaning
- later preservation
- interpretive cleanup
- worksheet procedure
- formula architecture
